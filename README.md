# -
Проект посвящен анализу статистических данных о смертности в России (2013–2023). Выполнена структуризация данных по регионам, причинам смерти и типам поселений (город/село). Рассчитаны суммарные, средние и медианные показатели, выявлены ключевые тренды и региональные различия в динамике смертности.
